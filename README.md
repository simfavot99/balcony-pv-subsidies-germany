# balcony-pv-subsidies-germany
Estimating the impact of subsidy programs on balcony PV adoption in Germany using heterogeneity-robust DiD.

## Shiny apps

I have built interactive Shiny apps to explore the data behind this project.

- **All municipalities** — interactive map of every German municipality + daily time-series of balcony-PV registrations per 100,000 inhabitants. See the [app tutorial / README](Public/Shiny%20apps/All%20Municipalities/README.md).
